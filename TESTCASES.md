# Тест-кейсы для API объявлений

**Host:** [https://qa-internship.avito.com](https://qa-internship.avito.com)


### TC-1: Создание объявления (Позитивный сценарий)

| Поле | Значение |
| :--- | :--- |
| **ID** | TC-1 |
| **Название** | Создание объявления (Позитивный сценарий) |
| **Приоритет** | High |
| **Цель** | Проверить создание объявления с корректным заполнением всех полей |
| **Тестовые данные** | `{<br> "sellerID": 112233,<br> "name": "Test Item",<br> "price": 500,<br> "statistics": {<br> "likes": 1,<br> "viewCount": 10,<br> "contacts": 2<br> }<br>}` |
| **Шаги** | 1. Подготовить тело запроса с валидными данными<br>2. Отправить `POST /api/1/item` |
| **Ожидаемый результат** | **200 OK**<br>Статус "Сохранили объявление"<br>Возвращается ID созданного объявления |


### TC-2: Создание объявления без обязательного поля (Негативный)

| Поле | Значение |
| :--- | :--- |
| **ID** | TC-2 |
| **Название** | Создание объявления без обязательного поля |
| **Приоритет** | Medium |
| **Цель** | Проверить валидацию обязательных полей |
| **Тестовые данные** | Тело запроса без поля `price` |
| **Шаги** | 1. Подготовить тело без поля `price`<br>2. Отправить `POST /api/1/item` |
| **Ожидаемый результат** | **400 Bad Request**<br>Объявление не создаётся |


### TC-3: Получение объявления по ID

| Поле | Значение |
| :--- | :--- |
| **ID** | TC-3 |
| **Название** | Получение объявления по ID |
| **Приоритет** | High |
| **Цель** | Проверить получение данных о конкретном объявлении |
| **Предусловия** | Объявление создано, ID известен |
| **Шаги** | 1. Отправить `GET /api/1/item/{id}` |
| **Ожидаемый результат** | **200 OK**<br>Поля `name`, `price`, `sellerId` совпадают с отправленными при создании |


### TC-4: Получение объявления по несуществующему ID

| Поле | Значение |
| :--- | :--- |
| **ID** | TC-4 |
| **Название** | Получение по несуществующему ID |
| **Приоритет** | Medium |
| **Цель** | Проверить обработку запроса с валидным, но отсутствующим ID |

| **Шаги** | 1. Отправить `GET /api/1/item/{uuid}` |

### TC-5: Получение всех объявлений продавца

| Поле | Значение |
| :--- | :--- |
| **ID** | TC-5 |
| **Название** | Получение всех объявлений продавца |
| **Приоритет** | High |
| **Цель** | Проверить получение списка объявлений по `sellerID` |
| **Предусловия** | Существует продавец с объявлениями |
| **Шаги** | 1. Отправить `GET /api/1/{sellerID}/item` |
| **Ожидаемый результат** | **200 OK**<br>В ответе приходит массив объявлений<br>Созданное объявление присутствует в списке |
