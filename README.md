# Тестовое задание для QA (Avito Start 2025)

**Задания 2.1**: автоматизация тестирования API микросервиса объявлений.

## Структура проекта

* **`test_api.py`** — основной файл с автотестами. Тесты написаны с учетом особенностей API (см. BUGS.md).
* **`conftest.py`** — файл конфигурации Pytest и фикстур для генерации валидных тестовых данных.
* **`requirements.txt`** — список необходимых библиотек Python.
* **`TESTCASES.md`** — описание сценариев проверки.
* **`BUGS.md`** — отчет о найденных багах и несоответствиях API.

## Стек технологий
* **Python 3.13**
* **Pytest** — фреймворк для запуска тестов.
* **Requests** — библиотека для HTTP-запросов.
